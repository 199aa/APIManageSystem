# API 可视化管理系统 - 前端

基于 Vue2 + Element UI 构建的 API 管理系统前端应用。

## 技术栈

- Vue 2.6
- Vue Router 3.x
- Vuex 3.x
- Element UI 2.x
- Axios

## 项目设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 生产环境构建
npm run build

# 代码检查
npm run lint
```

## 功能特性

- API 接口管理
- 接口文档可视化
- 响应式布局，适配多终端
- 用户权限管理

## 项目结构

```
src/
├── assets/          # 静态资源
├── components/      # 公共组件
├── views/          # 页面视图
├── router/         # 路由配置
├── store/          # Vuex状态管理
├── api/            # API请求封装
├── utils/          # 工具函数
└── App.vue         # 根组件
```
